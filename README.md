# 🐾 Happy Paws - Smart Pet Care Platform

Happy Paws is a modern web application designed for pet owners, featuring an **AI-powered Chatbot** and a seamless **Service Booking** system.

### 🚀 Key Features
- **AI Talk:** Real-time pet care advice powered by **Google Gemini API**.
- **Service Booking:** User-friendly interface to schedule pet services.
- **Lost & Found:** Dynamic matching logic to help find lost pets.
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop using HTML5 and CSS3.

### 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **AI Integration:** Google Gemini API
- **Tools:** Git, GitHub, VS Code
- **Training:** Professionally guided by **Techocean Hub**
